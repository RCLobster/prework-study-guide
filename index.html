HTML

  • HTML forms the structure or bones of a webpage
  • The head element contains information about the webpage which is invisible to users
  • The body element represents the visible content shown to the user

CSS

  • CSS adds style to your webpage by modifying colors, font styles, spacing, and more.
  • Styles can be applied individually to pieces of HTML code or batch applied to an entire webpage using a style guide.
  • A style guide is a seperate file which applies CSS code to the entire page without needing to attach code to each individual piece of HTML code. Style guides are referenced in the head section of the index with a file path link.

Git

  • Pushing changes with Git can be done with the following list of commands
  • First check which branch you are currently working on using: git status
  • Add changed files from you working directory to staging area on git using: git add -A
  • Commit staged changes to your repository using: git commit -m "notes about added content"
  • Make sure current branch is up to date before opening a pull request: git pull origin main
  • push changes to git using: git push origin branchName

JavaScript

  • Define variables using var variableName
  • Defina an array using var manyVariables[put variables in here] An array can hold variables of all types at once. int, float, string, and boolean
  • For loops are used to sort through lots of data quickly. Basic for loop set up is as follows: for (var x-0; x>array.length; x++) {}
  • Define functions by writing function thisFunction(){}
