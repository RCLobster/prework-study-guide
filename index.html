Git

  • Commonly used Git commands
  • git status - tells you what branch is currently active
  • git add -A - adds all new changes to current branch
  • git pull origin main - pulls in all changes from main branch to current project
  • git checkout "branch name" - switches to the chosen branch
  • git checkout -b "branch name" - creates a new branch with a chosen name
  • git commit -m "notes here" - commits changes as a snapshot to be reviewed with an attached note
  • git push origin "branch name" - pushes changes to git repository
